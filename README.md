# Zomato Clone

A Zomato-style restaurant listing page built with React and Tailwind CSS.

## Live Demo

[View on Vercel](https://your-project.vercel.app)

## Features

- Restaurant listing with search, filters, and categories
- Veg mode toggle with modal
- Skeleton loading
- Responsive design

## Tech Stack

- React
- Tailwind CSS
- Heroicons
- React Icons

## Run Locally
```bash
npm install
npm start
```

## Build
```bash
npm run build
```
